###变量

* 成员变量

	在类体中所定义的变量称为成员变量，成员变量在整个类中都有效。
	
	类的成员变量有分为静态变量和实例变量。
	
* 局部变量

	位于方法内部定义，在`{}`之间的	声明的变量。
