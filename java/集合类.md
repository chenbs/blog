#集合类
====

##Collection接口
Collection接口的常用方法：

* add(E e)
* remove(Object o)
* isEmpty()
* iterator()
* size()

##List集合

继承自Collection接口。

List接口的实现类常有:ArrayList和LinkedList

	List<E> list = new ArrayList<>();
	List<E> list2 = new LinkedList<>();
	
##Set集合

继承自Collection接口。

Set接口常见实现类有：HashSet类和TreeSet类。

##Map集合
	
没有继承自Collection接口。	

Map接口常见实现类有：HashMap类和TreeMap类。
